# 发布检查清单

在发布 CryptoPanel 到 Chrome Web Store 之前，请完成以下检查清单。

## 📦 构建和打包

- [ ] 更新 `manifest.json` 中的版本号
- [ ] 运行 `npm run build:extension` 构建成功
- [ ] 检查 `dist/` 目录包含所有必需文件
- [ ] 创建 ZIP 文件（不包含 dist 文件夹本身）
- [ ] 测试 ZIP 文件可以正常加载到 Chrome

## 🎨 视觉资源

- [ ] 图标已准备（16x16, 48x48, 128x128 PNG）
- [ ] 图标清晰可见，符合设计规范
- [ ] 至少 1 张截图已准备（1280x800 或更大）
- [ ] 截图展示了主要功能
- [ ] 所有截图清晰且专业

## 📝 文案内容

- [ ] 简短描述已准备（132 字符以内）
- [ ] 详细描述已准备（200-500 字）
- [ ] 描述准确反映扩展功能
- [ ] 关键词已优化
- [ ] 更新说明（Release Notes）已准备

## 🔒 隐私和安全

- [ ] 隐私政策已创建（`PRIVACY_POLICY.md`）
- [ ] 隐私政策已上传到可访问的 URL（GitHub Pages 等）
- [ ] 使用条款已创建（`TERMS_OF_SERVICE.md`，可选但推荐）
- [ ] 权限说明已完善
- [ ] 每个权限的用途已清楚说明

## 🧪 功能测试

- [ ] 所有主要功能正常工作
- [ ] 添加交易所配置功能正常
- [ ] 添加钱包地址功能正常

- [ ] 设置保存和加载正常
- [ ] 主题切换正常

- [ ] 错误处理正常

## 🌐 浏览器兼容性
