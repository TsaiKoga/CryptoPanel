**最后更新日期**：2024年12月

使用 CryptoPanel Chrome 扩展即表示您同意遵守以下使用条款。

## 服务说明

CryptoPanel 是一个加密货币资产看板工具，帮助用户统一管理 CEX 和链上资产。本扩展提供资产查询和展示功能，不涉及任何资金转移或交易操作。

   - 建议仅授予 API 密钥只读权限
   - 我们不对因密钥泄露造成的损失负责
   - 请妥善保管您的 API 密钥，不要分享给他人

   - 我们努力确保数据准确性，但不保证 100% 准确
   - 价格数据来自第三方 API，可能存在延迟或错误
   - 请以交易所和链上数据为准进行重要决策
   - 我们不对因数据不准确造成的损失负责

3. **合规使用**
