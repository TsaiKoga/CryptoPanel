# CryptoPanel 隐私政策

**最后更新日期**：2025年12月

## 数据收集

CryptoPanel 是一个本地运行的 Chrome 扩展，我们**不会收集、存储或传输任何用户数据到外部服务器**。

### 本地存储的数据

以下数据仅存储在您的浏览器本地：

1. **交易所 API 配置**
   - API Key
   - Secret Key
   - Passphrase（如适用）
   - 这些信息仅用于访问您授权的交易所账户

2. **钱包地址**
   - 您添加的加密货币钱包地址
   - 仅用于查询链上资产

3. **应用设置**
   - 语言偏好
   - 主题设置
   - 显示偏好（如隐藏小额资产）

4. **缓存数据**
   - 资产数据缓存（24小时）
   - 价格数据缓存（30分钟）

### 数据传输

扩展会向以下服务发送请求以获取数据：

1. **交易所 API**（仅当您配置了 API 密钥时）
   - Binance API：获取账户余额和资产信息
   - OKX API：获取账户余额和资产信息

2. **价格 API**（公开 API，无需认证）
   - DeFiLlama API：获取代币价格
   - CryptoCompare API：获取代币价格

3. **区块链 RPC 节点**（公开节点）
   - ThirdWeb、Infura、Alchemy 等：用于查询链上资产余额

**重要**：所有 API 密钥和敏感信息仅存储在您的浏览器本地，不会传输到我们的服务器或任何第三方服务器。

### 数据安全

- 所有数据使用 Chrome 的 `chrome.storage.local` API 存储
- 数据仅存在于您的设备上
- 无法访问您的数据
- 卸载扩展将删除所有本地数据

### 第三方服务

我们使用以下第三方服务：

- **DeFiLlama**：价格数据（公开 API）
- **CryptoCompare**：价格数据（公开 API）
- **RPC 节点提供商**：链上数据查询（ThirdWeb、Infura、Alchemy）

这些服务可能有自己的隐私政策，请查看其官方网站。

### 您的权利

- 您可以随时查看、修改或删除存储的数据
- 卸载扩展将永久删除所有本地数据
- 我们不收集任何个人身份信息
